# Job Application Tracker

A desktop application built with Python, Tkinter, and SQLite to help users track job applications.

## Features

* Add applications
* Edit applications
* Delete applications
* Search records
* Sort applications
* View statistics
* Export data to CSV
* Automatic date and time tracking
* Status-based color coding

## Technologies Used

* Python
* Tkinter
* SQLite
* CSV

## Future Improvements

* Excel export
* Follow-up reminders
* Application filtering by date
* Email integration
